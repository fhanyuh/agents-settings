# Iteration Log

One markdown file per "e"/"enhance" or "n"/"next"/"n{x}" run, written automatically
per `AGENTS.md` §2b. Filename pattern: `<YYYY-MM-DD-HHmm>-<trigger>.md`
(e.g. `2026-07-08-1430-n2.md`).

Each entry covers: what was requested, steps taken, what succeeded/failed, current
state after the iteration, and open considerations for next time. Written even when
an iteration fails or is only partially complete.
